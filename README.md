# xiaochen-skills

小陈的 Claude Code 技能集。

## Skills

| Skill | 说明 |
|-------|------|
| [davinci-subtitle-fixer](./skills/davinci-subtitle-fixer/) | 达芬奇字幕修正：Gemini + Claude 混合架构修正 ASR 字幕 |
| [wechat-publisher](./skills/wechat-publisher/) | 公众号排版发布：Obsidian Markdown → 微信公众号格式 |
| [wechat-mosaic](./skills/wechat-mosaic/) | 微信截图打码：批量马赛克 + 朋友圈文案生成 |
| [cc-price-calculator](./skills/cc-price-calculator/) | 订阅成本计算：Claude 订阅/拼车/中转成本计算器 |

## 安装

在 Claude Code 中运行：

```
/plugin marketplace add chenhuajinchj/xiaochen-skills
```
