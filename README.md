# xiaochen-skills

小陈的 Claude Code 技能集。

## Skills

| Skill | 说明 |
|-------|------|
| [davinci-subtitle-fixer](./plugins/cyxj-davinci-subtitle-fixer/) | 达芬奇字幕修正：Gemini + Claude 混合架构修正 ASR 字幕 |
| [wechat-publisher](./plugins/cyxj-wechat-publisher/) | 公众号排版发布：Obsidian Markdown → 微信公众号格式 |
| [wechat-mosaic](./plugins/cyxj-wechat-mosaic/) | 微信截图打码：批量马赛克 + 朋友圈文案生成 |
| [cc-price-calculator](./plugins/cyxj-cc-price-calculator/) | 订阅成本计算：Claude 订阅/拼车/中转成本计算器 |
| [obsidian-wiki-compiler](./plugins/cyxj-obsidian-compiler/) | 知识库编译：基于 Karpathy LLM Wiki 方法论的三层知识架构编译 |
| [poster-design](./plugins/cyxj-poster-design/) | 海报设计：一句话生成大师级海报、书籍封面、专辑封面（33+ 设计师风格） |
| [youtube-topic-discovery](./plugins/cyxj-youtube-topic-discovery/) | YouTube 选题发现：搜索最近 48 小时新视频，去重聚类写入 Obsidian |
| [notebook-research](./plugins/cyxj-notebook-research/) | Notebook LM 研究：将选题视频提交 Notebook LM，拉取转录稿和报告 |

## 安装

在 Claude Code 中运行：

```
/plugin marketplace add chenhuajinchj/xiaochen-skills
```
